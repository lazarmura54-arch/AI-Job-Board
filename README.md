# AI Job Board

## Overview

AI Job Board is a modern web application developed using Flask, HTML, CSS, and JavaScript. It allows users to browse software engineering jobs, search and filter listings, and view detailed job information.

## Features

- Home Page
- Responsive UI
- Search Jobs
- Filter Jobs
- Job Details
- Save Jobs
- Apply Button
- Company Logos
- Skills Tags
- Professional Layout

## Technologies

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- GitHub Actions
- Vercel

## Installation

```bash
pip install -r requirements.txt
python app.py
```
